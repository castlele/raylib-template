# Game template

Template for quick application development using [raylib](https://github.com/raysan5/raylib).

## How To Run

### Web Assembly

Prerequisites:
- [emscripten](https://emscripten.org/docs/getting_started/downloads.html)

Run:
```bash
make serve PLATFORM=PLATFORM_WEB
```

### Desktop

Run:
```bash
make run
```
